# Smart Factory Demo

Applicazione WEB sviluppata con **Vite**, **React**, **TypeScript** e **Tailwind CSS**.

Il progetto include la formattazione automatica del codice tramite **Prettier**.

---

## Tecnologie utilizzate

- Vite
- React
- TypeScript
- Tailwind CSS
- Prettier

---

## Requisiti

Installazioni obbligatorie:

- **Node.js** (versione consigliata: >= 18)
- **npm**

Verificare le versioni installate:

```bash
node -v
npm -v
```

## Passaggi installazione file Applicazione

1. Entrare nella cartella

```bash
2. install npm

3. npm run dev
```
